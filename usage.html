Grav CMS

Grab the modified smileys.css here, and replace the following file in your Grav directory:

users/plugins/smileys/assets/smileys.css

Extract the emote pack into here:

users/data/smileys/

In the Smileys plugin menu, select Pixel Emotes Mono or Chromatic, depending on which one you want to use.

WordPress

Taken from the WordPress Codex:

Upload the images you want with the same name to your server (say in wp-content/images/smilies) and put this in your theme's function.php:

add_filter( 'smilies_src', 'my_custom_smilies_src', 10, 3 );
function my_custom_smilies_src( $img_src, $img, $siteurl )
{
    return $siteurl.'/wp-content/images/smilies/'.$img;
}
